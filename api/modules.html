createService

  • createService(__namedParameters: { cogSettings: CogSettings; grpcURL: string; streamingTransportFactory?: grpc.TransportFactory; unaryTransportFactory?: grpc.TransportFactory }): CogmentService
    • Defined in cogment/Cogment.ts:61

    Creates a new CogmentService from a generated CogSettings. Optionally accepts transports used by gRPC clients.

    example

    Instantiating the cogment API.

    import {createService} from 'cogment';
import cogSettings from 'CogSettings';

const cogment = createService(cogSettings);

    Parameters

    • __namedParameters: { cogSettings: CogSettings; grpcURL: string; streamingTransportFactory?: grpc.TransportFactory; unaryTransportFactory?: grpc.TransportFactory }
      • cogSettings: CogSettings

        Settings loaded from a generated CogSettings file.

      • grpcURL: string

        HTTP(S) url of grpc-web reverse proxy to orchestrator. Defaults to //${window.location.hostname}:8080

      • Optional streamingTransportFactory?: grpc.TransportFactory

        A grpc.TransportFactory used to instantiate streaming connections to the backend. Defaults to grpc.WebsocketTransport().

      • Optional unaryTransportFactory?: grpc.TransportFactory

        A grpc.TransportFactory used to make unary (non-streaming) requests to the backend. Defaults to grpc.CrossBrowserHttpTransport({withCredentials: false}).

    Returns CogmentService
